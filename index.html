常见问题

解答您的疑问，帮助您更好地使用平台

网站如何计算费用？

本站基于模型分词结果进行计费。对流式请求，由于接口可能不会实时返回消耗量，服务器会使用 Tokenizer，并根据不同模型选择对应编码器（例如 gpt-3.5 / gpt-4 系列使用 cl100k_base；text-davinci-003 使用 p50k_base）进行精确估算。计费时会分别统计输入与输出 token，并按对应倍率扣费；对非流式请求，则以接口返回的 usage 为准。

为什么余额可能出现负数？余额为负会怎样？

由于模型输出长度无法完全预估，在极少数情况下可能出现余额短暂为负的情况。余额为 0 或负数时，系统会通知你并限制部分服务（包括部分网页功能与 API 调用）。建议及时充值以恢复正常使用。

什么是 VIP（合作用户）？如何升级？

VIP（合作用户）用于提供更稳定的资源保障与更优的充值/计费倍率。你当前用户等级为 default；当累计充值达到 500 USD 后，可联系管理员升级为 VIP。VIP 用户通常享有更宽松的速率限制与更优先的模型资源。

什么是专线（Direct - 直连）？

Direct 专线面向用量稳定的团队用户。开通后你将独占直连线路，不与其他用户共享 RPM，以获得更稳定的吞吐与更确定的延迟表现。具体开通门槛与方案请联系客户支持。

如何开票？

支持开票。充值金额计入账户余额；如需开票将另加税点 1%。请联系客户支持获取流程与材料清单。
